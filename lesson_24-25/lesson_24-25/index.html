Задание
Ответ

1) Сначала находим в документе элемент, сожержащий данный data атрибут с помощью метода querySelector() объекта document, в аргументе метода передаем значение атрибута в квадратных скобках обернутых в кавычки (document.querySelector("[data-say-hi]")) и полученный элемент присваиваю переменной elementWithDataAttr. Для получения значения атрибута можно использовать свойство dataset, так как атрибут начинается с приставки data и через точку прописываю имя атрибута. Так как имя состоит из 2-х слов используй следующую запись (elementWithDataAttr.dataset.sayHi). Значение атбируба также можно получить с помощью метода getAttribute(), а в аргумент метода передать полное название атрибута в кавычках elementWithDataAttr.getAttribute("data-say-hi"), строки 46-48
